# Environment variables

Every environment variable the code reads, found by scanning the sources for
`env::var` calls and `RIVEN_*` literals. The table is regenerated by
`cargo run --bin gen-docs`, so a knob added in code shows up here without
anyone remembering to list it — `make docs-check` fails when it is missing.

Everything here has a working default. None of it needs to be set for Riven
to run; these are the levers for when a deployment needs different bounds
than the defaults give it. Settings exposed through plugins are documented
under [plugins/](plugins/README.md) instead, with their own
`RIVEN_PLUGIN_SETTING__*` names.
