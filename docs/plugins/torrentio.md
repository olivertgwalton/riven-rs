# Torrentio

Scraper that queries [Torrentio](https://torrentio.strem.fun) for torrents. Like
[Comet](comet.md) it is a Stremio addon, so results come back matched to the IMDb ID
being scraped.

The `filter` setting is passed through to Torrentio verbatim as a URL path segment. It
uses `|` between clauses and takes the same options as the Torrentio configuration page,
so the fastest way to build one is to configure the addon in a browser and copy the
segment out of the resulting URL. The default sorts by quality-then-size and drops 3D,
480p, screener, and cam releases.
