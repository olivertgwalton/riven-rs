# Plex

Media server integration. On a successful download it refreshes the Plex library section
containing the new file, and it reports active playback sessions back to the core so the
VFS knows which files are in use.

The token is a Plex `X-Plex-Token`, not your account password — the
[Plex support article](https://support.plex.tv/articles/204059436) covers how to obtain
one. `plexserverurl` must be reachable from the Riven container, which usually means the
container name or LAN address rather than `localhost`.

Library sections are cached for five minutes, so a section added in Plex can take that
long to become visible to Riven.
