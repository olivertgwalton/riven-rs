# Newznab

Usenet indexer search. Queries every configured [Newznab](https://newznab.readthedocs.io)
indexer in parallel and deduplicates the results, so NZBGeek, NZBFinder, a Prowlarr
endpoint, and anything else speaking the protocol can all be searched at once.

Indexers are configured as a dictionary — each entry gets its own URL, API key, and
category list. Categories default to `2000,5000` (movies and TV); leave them alone unless
your indexer uses non-standard IDs.

This plugin only *finds* NZBs. Downloading them requires either [usenet](usenet.md) for
direct NNTP streaming, or [stremthru](stremthru.md) with `newznabenabled` turned on.
